🎨 Colorful Calculator (HTML, CSS, JavaScript)
📌 Project Description

This project is a Colorful Web Calculator created using HTML, CSS, and JavaScript.
It performs basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage.

The calculator has a modern colorful design with responsive buttons and a clean user interface.
🌟 Live Demo

🔗 Live Site: Add your deployed link here 📦 Repository:http://127.0.0.1:3000/index.html

🚀 Features

Basic arithmetic operations (+, −, ×, ÷)

Percentage calculation (%)

Clear button (C)

Delete last digit (DEL)

Colorful modern UI design

Responsive button layout

Simple and lightweight web application

🛠 Technologies Used

HTML5 – structure of the calculator

CSS3 – styling and layout

JavaScript – calculator functionality

📂 Project Structure
calculator-project/
│
├── index.html      # Main calculator structure
├── style.css       # Styling and layout
└── script.js       # Calculator logic
▶️ How to Run the Project

Download or clone the repository

Open the project folder

Double click index.html

The calculator will open in your web browser

🧮 Calculator Functions
Button	Function
C	Clear display
DEL	Delete last number
%	Percentage
+ - * /	Arithmetic operations
=	Calculate result
